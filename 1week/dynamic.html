Day 1 - HTML 기초

HTML의 기본 구조와 태그에 대해 학습합니다.

  • HTML 문서 구조 이해
  • 기본 태그들 (h1, p, div, span 등)
  • 시맨틱 HTML 요소
  • 폼 요소와 입력 필드

오늘은 웹 개발의 기초가 되는 HTML에 대해 배워보겠습니다.

Day 2 - CSS 스타일링

CSS를 사용한 웹페이지 스타일링을 학습합니다.

  • CSS 선택자와 속성
  • 박스 모델과 레이아웃
  • Flexbox와 Grid
  • 반응형 디자인

아름다운 웹페이지를 만들기 위한 CSS의 핵심 개념들을 익혀보겠습니다.

Day 3 - JavaScript 기초

JavaScript 프로그래밍의 기본을 학습합니다.

  • 변수와 데이터 타입
  • 함수와 스코프
  • 조건문과 반복문
  • DOM 조작

웹페이지에 동적인 기능을 추가하는 JavaScript의 기초를 배워보겠습니다.

Day 4 - 이벤트 처리

JavaScript 이벤트 처리와 상호작용을 학습합니다.

  • 이벤트 리스너
  • 마우스와 키보드 이벤트
  • 폼 이벤트 처리
  • 이벤트 버블링과 캡처링

사용자와의 상호작용을 처리하는 이벤트 시스템을 익혀보겠습니다.

Day 5 - API와 비동기 처리

외부 데이터와의 통신과 비동기 프로그래밍을 학습합니다.

  • Fetch API 사용법
  • Promise와 async/await
  • JSON 데이터 처리
  • 에러 핸들링

실제 데이터를 가져와서 웹페이지에 표시하는 방법을 배워보겠습니다.

Day 6 - 로컬 스토리지

브라우저의 로컬 스토리지를 활용한 데이터 저장을 학습합니다.

  • localStorage와 sessionStorage
  • 데이터 저장과 불러오기
  • JSON 직렬화/역직렬화
  • 데이터 영속성 관리

사용자 데이터를 브라우저에 저장하고 관리하는 방법을 익혀보겠습니다.

Day 7 - 프로젝트 완성

지금까지 배운 내용을 종합하여 완성된 프로젝트를 만들어봅니다.

  • 프로젝트 구조 설계
  • 코드 리팩토링
  • 성능 최적화
  • 배포 준비

1주일 동안 배운 모든 기술을 활용하여 완성도 높은 웹 애플리케이션을 만들어보겠습니다.
